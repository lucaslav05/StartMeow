# StartMeow

Project Starter with a Terminal UI Made For BCIT HackTheBreak 2026.

## Quick Start

Download the latest executable directly from the [GitHub Releases](https://github.com/lucaslav05/StartMeow/releases) page, move it to your project folder and run it:

```bash
mv StartMeow ~/newProject
cd ~/newProject
./StartMeow
```

No build step required.

---

## Building from Source

Clone the repository and build the binary:

```bash
git clone https://github.com/lucaslav05/StartMeow
cd StartMeow
go build
```

This produces an executable called `StartMeow` in the current directory.

## Running

Move and run the executable in the folder that will contain the project:

```bash
# move to a project folder
mv StartMeow ~/newProject

# cd into the new project folder
cd ~/newProject

# run the binary
./StartMeow
```

## Requirements

- Go 1.21 or later (only needed if building from source)
